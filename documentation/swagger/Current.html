REST api is self documented using Swagger annotations. Access documentation via this url http://localhost:8080/swagger-ui.html

REST api are available in com.salesmanager.shop.store.api.v1 package

Public api #

Public api are available for POST and GET methods without being authenticated.

Private api #

Private api will require authentication. Authentication method is based on JWT / Bearer token. url pattern for api requiring authentication is /api/v1/private/*
